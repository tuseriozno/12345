### Позиционные аргументы
По умолчанию, когда вы создаете функцию в Python, она сохраняет порядок аргументов в определении функции.

Например. В функции ниже первый входной аргумент всегда будет выводиться до второго. `a = 2, b = 1`

```
def my_function(a, b)
  print(a)
  print(b)
  
 my_function(2, 1)
 # Он напечатает:
 # 2
 # 1
```

### Аргументы-ключевые слова
Вы можете использовать ключевые слова, когда предоставляете аргументы при вызове функции, чтобы сократить путаницу, какое значение присваивается какому входному параметру.
