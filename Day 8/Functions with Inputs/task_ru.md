Ранее мы видели, что функции позволяют нам упаковать код в блок с названием, который можно использовать повторно в дальнейшем.

### ПАУЗА 1 - Обзор
- Создайте функцию с названием `greet()`.
- Напишите внутри функции 3 выражения `print`.
- Вызовите функцию `greet()` и запустите ваш код.

### Ввод
Добавив имя переменной в скобки при создании (определении) новой функции, мы даем возможность этой функции принимать вводные данные при вызове.

Это означает, что мы можем изменять поведение функции каждый раз, передавая разные входные данные.

```
# Создание функции
def myFunction(input):
    print(f"Привет! {input}")
```
```
# Использование функции
myFunction("Томми") 
# Выведет "Привет! Томми"
```

### Вводные данные являются переменными
Когда вы создаете функцию с вводными данными, вы определяете имя переменной, которое будет присвоено переданным данным.

Имя входной переменной, например, `name` в этом коде: `def greet(name):` называется параметром.

Значение входной переменной, например, `Анжела` при вызове предыдущей функции `greet`: `greet("Анжела")` называется аргументом.
