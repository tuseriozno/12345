Циклы позволяют нам сказать компьютеру повторять действия, не записывая один и тот же код многократно. Если бы мы захотели, чтобы компьютер напечатал числа от 1 до 100, было бы очень утомительно писать оператор `print` для каждого числа или даже просто перечислять все числа от 1 до 100. Циклы позволяют нам создать правило, которое компьютер может использовать для выполнения повторяющегося действия.

### Синтаксис

```
for <имя переменной для каждого элемента> in <Список>:
    <выполнить что-то>
    <выполнить что-то еще> 
```

### ПАУЗА 1 - Будь компьютером
Предскажите, что будет напечатано из следующего кода:

```
fruits = ["Apple", "Peach", "Pear"]
for fruit in fruits:
    print(fruit)
    print(fruit + " pie")
```

### Отступы
Отступы имеют большое значение в программировании на языке Python. Каждый раз, когда вы видите символ `:`, нужно быть внимательным к отступам, которые следуют за ним.

Например, этот код будет работать совсем иначе:

```
fruits = ["Apple", "Peach", "Pear"]
for fruit in fruits:
    print(fruit)
    print("Hello")
```

чем этот код:

```
fruits = ["Apple", "Peach", "Pear"]
for fruit in fruits:
    print(fruit)
print("Hello")
