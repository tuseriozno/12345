Сочетание функции `range()` с циклом For в Python позволяет нам выполнить цикл столько раз, сколько мы хотим. Вместо прохода через каждый элемент в списке, мы можем проходить через диапазон чисел.

### Функция Range

`range(1, 10)`

Этот код сам по себе ничего не делает. Например, если вы попытаетесь распечатать его, он не выдаст вам отдельные числа.

Но он может использоваться в сочетании с циклами For. например:

``` 
for number in range(1, 10):
    print(number)
```

Это выведет каждое из чисел от 1 до 9. Так что диапазон также может быть выражен так:

`a <= range(a, b) < b`

Где диапазон чисел включает в себя нижнюю границу, но не включает верхнюю границу.
