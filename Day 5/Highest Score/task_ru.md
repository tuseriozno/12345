### Сумма
Python имеет множество встроенных функций для работы с числами. Одна из них помогает нам вычислять сумму (или итоговое значение).
Например.
```
student_scores = [180, 124, 165, 173, 189, 169, 146]
total_score = sum(student_scores) 
```
Но как работает `sum()` за кулисами? Код написан людьми, разработавшими Python, и он может выглядеть примерно так:

```
student_scores = [180, 124, 165]

sum = 0
for score in student_scores:
    sum += score
    
print(sum)
```

### ПАУЗА 1 - Максимум
В Python также существуют встроенные методы `max()` и `min()`, которые позволяют вам передать список чисел, и они выдадут вам наибольшее число или наименьшее число.

Ваша задача - понять, как программисты Python могли бы создать эту функциональность с использованием циклов и условий.

## ВЫПОЛНИТЕ ЭТО ЗАДАНИЕ, НЕ ИСПОЛЬЗУЯ max()

Вам дан список оценок за экзамен, и вы должны напечатать самую высокую оценку из списка.
Вы должны использовать то, что вы узнали о списках, циклах for и условиях, чтобы напечатать самую высокую оценку в списке student_scores.
Например, если оценки были:
```
8 65 89 86 55 91 64 89
```
Ваш код должен выводить
```
91
```
