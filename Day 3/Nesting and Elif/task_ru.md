### Вложенные условные операторы
Вы можете поместить операторы if/else внутрь других операторов if/else. Это называется вложенностью.
например:

```
if maths_score >= 90:
    if english_score >= 90:
        print("Вы хорошо справляетесь со всем")
    else:
        print("Вы хорошо справляетесь с математикой")
if english_score >= 90:
    print("Вы хорошо знаете английский")
```

В этом случае, только когда у студента более 90 баллов по математике и английскому языку, они получают сообщение "Вы хорошо справляетесь со всем".

Примечание: Вы также можете иметь операторы if, которые не имеют сопряженного оператора else.
