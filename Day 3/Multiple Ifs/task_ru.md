Вы можете написать столько операторов if, сколько вам нужно, чтобы проверить различные условия, которые не связаны друг с другом.
Сравните блоки кода ниже:
```
# Если/иначе если/в противном случае
if <условие 1 верно>
    <выполнить А>
elif <условие 2 верно>
    <выполнить В>
else
    <выполнить С>
```
```
# Вложенные операторы if
if <условие 1 верно>
    <выполнить А>
    if <условие 2 верно>
        <выполнить В>
        if <условие 3 верно>
            <выполнить С>
```

```
# Несколько операторов if
if <условие 1 верно>
    <выполнить А>
if <условие 2 верно>
    <выполнить В>
if <условие 3 верно>
    <выполнить С>
```

В блоке если/иначе если/в противном случае может произойти только одно из событий А, В или С, потому что они взаимоисключающие. Первое условие должно быть не выполнено, чтобы перейти к следующему, и если все условия не выполнились, то выполняется else. Условие 2 проверяется только в том случае, если условие 1 ложно.

Во вложенных операторах if, могут произойти все события A, B и C, но условия 1, 2 и 3 все должны быть истинны. Компьютер проверит условие 2 только в случае, если условие 1 истинно.

В случае с несколькими операторами if, также могут произойти все события A, B и C. Но они полностью независимы друг от друга. Событие C может произойти, даже если события A и B не произошли. Каждое условие проверяется, независимо от результата других.
