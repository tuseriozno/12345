### Проверка Условия
Научитесь использовать условные операторы в Python для проверки условий и указания компьютеру, что делать в каждом случае. 
например:

`if <данное условие верно>:`

&nbsp;&nbsp;&nbsp;&nbsp;`<тогда выполнить данную строку кода>`

### Что если условие ложное?
Ключевое слово else используется для определения блока кода, который будет выполняться, если условие, проверенное в операторе if, ложно.

`if свиньи могут летать:`

&nbsp;&nbsp;&nbsp;&nbsp;`<Некоторый код, который никогда не выполнится>`

`else:`

&nbsp;&nbsp;&nbsp;&nbsp;`print("Это реальная жизнь")`

### Отступы в Python
Поймите важность использования отступов в Python как способа делать определенные строки кода подчиненными другим строкам кода.

например:

`if 5 > 2: #Это родительская строка кода`

&nbsp;&nbsp;&nbsp;&nbsp;`print("да") #это дочерняя строка кода`

### Операторы Сравнения
* `>` Больше
* `<` Меньше
* `>=` Больше или равно
* `<=` Меньше или равно
* `==` Равно
* `!=` Не равно
