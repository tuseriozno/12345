Вы можете использовать docstrings для написания многострочных комментариев, которые документируют ваш код.

### Синтаксис
Просто поместите свой текст внутрь пары из трех двойных кавычек.

например,
```
"""
Мой
Многострочный
Комментарий
"""

```

### Документирование функций
Одной из удобных особенностей docstrings является то, что вы можете использовать его сразу после определения функции и этот текст будет отображаться при наведении на вызов функции. Это хороший способ напомнить себе, что делает самодельная функция.

например,
```
def my_function(num):
    """Умножает число на самого себя."""
    return num * num
```
