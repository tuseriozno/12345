Мы видели функции, которые имеют только тело выполнения, функции с входными данными, которые позволяют варьировать выполнение тела функции, и теперь мы увидим окончательную форму функций. Функции, которые могут иметь результаты.

### ПАУЗА 1
Создайте функцию под названием `format_name()`, которая принимает два входных параметра: `f_name` и `l_name`.

### ПАУЗА 2
Используйте функцию `title()`, чтобы изменить параметры `f_name` и `l_name` в Заглавный Регистр.

### Синтаксис
Вы можете создать функцию с телом, входом и выходом следующим образом:

```
def имя_функции(входной_параметр):
    <тело функции, которое использует входной_аргумент>
    return вывод
```

### Вывод на печать vs Вывод
Return (возврат) vs Display (показ): оператор return используется для возвращения значения из функции, которое можно использовать позже, в то время как print используется только для отображения значения в консоли, которое может видеть только программист.
