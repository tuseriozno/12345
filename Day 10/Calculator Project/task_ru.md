### Сохранение функций как значения переменной
Вы можете сохранить ссылку на функцию в качестве значения переменной.
например,
```
def add(n1, n2):
    return n1 + n2
    
    
my_favourite_calculation = add
my_favourite_calculation(3, 5)  # Вернёт 8
```
В начальном файле вы увидите словарь, который ссылается на каждое из математических расчетов, которые может выполнить наш калькулятор. Попробуйте его и посмотрите, сможете ли вы сделать сложение, вычитание, умножение и деление.

### Функциональность
- Программа просит пользователя ввести первое число.
- Программа просит пользователя ввести математический оператор (на выбор "`+`", "`-`", "`*`" или "`/`")
- Программа просит пользователя ввести второе число.
- Программа вычисляет результат на основе выбранного математического оператора.
- Программа спрашивает, хочет ли пользователь продолжить работу с предыдущим результатом.
- Если да, программа зацикливается на использовании предыдущего результата в качестве первого числа, а затем повторяет процесс расчета.
- Если нет, программа запрашивает у пользователя первое число снова и забывает все предыдущие расчеты.
