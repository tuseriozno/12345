Функции завершаются при использовании ключевого слова `return`. Если вы пишете код ниже оператора return, этот код не будет выполнен.

Однако, вы можете иметь несколько операторов return в одной функции. Но как это работает?

### Условные возвраты

Когда у нас есть управление потоком, то есть код ведет себя по-разному (идет по разным путям выполнения) в зависимости от определенных условных проверок, мы можем получить несколько вариантов окончания (возвратов).

например:
```
def canBuyAlcohol(age):
    if age >= 18:
        return True
    else:
        return False
```

### Пустые возвраты
Вы также можете написать return без чего-либо после, и это просто говорит функции о выходе.

например:
```
def canBuyAlcohol(age):
    # Если тип данных ввода возраста не является целым числом, тогда выйти.
    if type(age) != int:
        return

    if age >= 18:
        return True
    else:
        return False
```
