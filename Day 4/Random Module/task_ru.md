### Генераторы псевдослучайных чисел
Компьютеры не являются случайными, потому что они созданы из цепей и переключателей. Но случайность очень важна при разработке программного обеспечения, особенно игр. Было бы очень скучно, если бы каждое движение в видеоигре предопределялось.

Таким образом, несколько ученых в области компьютерных наук изобрели генераторы псевдослучайных чисел, например: https://en.wikipedia.org/wiki/Mersenne_Twister.

Если вы хотите узнать больше о генераторах псевдослучайных чисел, рекомендую вам посмотреть это видео от Khan Academy: https://www.youtube.com/watch?v=GtOt7EBNEwQ&ab_channel=KhanAcademyLabs

### Модуль random в Python
Документацию можно прочитать здесь:  
https://docs.python.org/3/library/random.html  

Чтобы использовать его, сначала нужно импортировать модуль:

`import random`

### Случайные целые числа в диапазоне

```
import random
rand_num = random.randint(1, 10)

# Это создаст случайное целое число в диапазоне от 1 до 10 (включительно).
```

Python позволяет нам размещать код в различных файлах и импортировать этот код при необходимости. Это означает, что мы можем лучше организовать и структурировать наш код.

### Случайные числа с плавающей точкой
Вы можете сгенерировать случайное число между 0.0 (включительно) и 1.0 (не включительно), используя следующий код из модуля random:

```
import random
rand_num_0_to_1 = random.random()
```
Это также может быть представлено так:

`0.0 <= random.random() < 1.0`

Вы можете расширить диапазон случайных чисел, генерируемых этим методом, используя умножение.

Например, `random.random() * 5`

Сгенерирует случайное число в диапазоне от 0 до 5.

Другой способ сгенерировать числа с плавающей точкой — использовать функцию `uniform()`.

```
import random
random_float = random.uniform(1, 10)
# Это создаст случайное число с плавающей точкой в диапазоне от 1 до 10.
```
Метод может включать или не включать верхнюю границу диапазона в зависимости от округления числа с плавающей точкой.  
Таким образом, лучше всего это представлять так:

`a <= random.uniform(a,b) <= b`

В зависимости от того, хотите ли вы включить верхнюю границу, вы можете выбрать между `random.random()` и `random.uniform()`.
