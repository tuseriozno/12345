Функция в своем самом простом виде — это всего лишь имя-обертка для блока кода. Вы задаете ей имя, и затем, когда вызываете функцию по этому имени, весь код внутри блока функции будет выполнен. Это может помочь нам сэкономить время и сократить повторяющийся код.

### Определение новой функции
```
def <имя функции>():
    print("Hello")
    # Выполнить что-то еще
    # Выполнить что-то еще ...
```

### Вызов функции
Вызов функции просто означает запуск функции. Мы можем вызывать функцию в любой точке нашего кода на Python.

```
<имя функции>()
```

Объединим все вместе, например:
```
# Создание функции
def get_user_name():
    name = input("Как вас зовут? ")
    print("Привет, " + name)
    # Внутри функции

# Вне функции
print("Привет")
get_user_name() # Вызов функции
```

Этот код приведет к следующей последовательности вывода:
```
Привет
Как вас зовут? # Я ввожу Ангела
Привет
Ангела
