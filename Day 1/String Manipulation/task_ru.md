Учитесь использовать конкатенацию строк и последовательность escape для новой строки, чтобы форматировать строки в Python.

### ПАУЗА 1. Используйте `\n`, чтобы добавить ещё одну строку "Привет, мир".

Так что итоговый вывод выглядит так:

Привет, мир!

Привет, мир!

Привет, мир!

### ПАУЗА 2. Добавьте пробел между строками

Таким образом, между строкой `Привет` и `Анжела` есть пробел, когда запускается оператор печати.

Вывод должен выглядеть так:

Привет Анжела
