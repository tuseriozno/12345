Словарь в Python работает аналогично словарю в реальной жизни. Это структура данных, которая позволяет связывать ключ с значением и объединять эти два элемента данных вместе.

Вот как вы создаете словарь в Python:
```
# Пример словаря
colours = {
   "apple": "red", 
   "pear": "green", 
   "banana": "yellow"
}

```

Вот так вы получаете элементы из словаря:
```
print(colours["pear"])
#Напечатает "green"
```

Вот как создать пустой словарь:
```
my_empty_dictionary = {}
```

Вот как вы можете добавить новые элементы в существующий словарь:

```
colours["peach"] = "pink"
```

Вот как вы можете отредактировать существующее значение в словаре:
```
colours["apple"] = "green"
```

Вот как пройтись по словарю и распечатать все ключи:
```
for key in colours:
    print(key)
```

Вот как пройтись по словарю и распечатать все значения:
```
for key in colours:
    print(colours[key])
```
