Вы можете комбинировать различные типы данных, чтобы достичь желаемой структуры.

### Вложение списка в словарь
Вместо строки, присвоенной ключу, мы можем заменить ее на список. Вы можете вложить список в словарь следующим образом:

```
my_dictionary = {
    key1: [Список],
    key2: Значение,
}
```

### ПАУЗА 1
Попробуйте выяснить, как вывести на печать "Lille" из вложенного списка под названием `travel_log`.
```
travel_log = {
    "France": ["Paris", "Lille", "Dijon"],
    "Germany": ["Stuttgart", "Berlin"],
}
```
<div class="hint">
  Чтобы получить эту часть: <code>["Paris", "Lille", "Dijon"]</code>
Выберите: <code>travel_log["France"]</code>

Следовательно, чтобы получить Lille, вам нужно:
<code>travel_log["France"][1]</code>
</div>

### Вложение списков в другие списки

Мы уже видели вложенные списки:

```
nested_list = ["A", "B", ["C", "D"]]
```

### ПАУЗА 2
Помните ли вы, как получить элементы, которые глубоко вложены в список? Попробуйте вывести на печать "D" из списка `nested_list`.

<div class="hint">
  Чтобы получить этот список: ["C", "D"] нам нужен код:

<code>nested_list[2]</code>

Следовательно, чтобы получить "D" нам нужно:

<code>nested_list[2][1]</code>
</div>


### Вложение словаря в словарь
Вы также можете вложить словарь в словарь:

```
my_dictionary = {
    key1: Значение,
    key2: {Ключ: Значение, Ключ: Значение},
}
```
