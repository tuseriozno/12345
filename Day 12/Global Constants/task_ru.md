Вы можете определить глобальные константы в своем файле кода для удобного доступа. Их задача - быть "установленными и забытыми", чтобы вы могли использовать их значения, но никогда не нужно было их изменять.

### Конвенция именования
Глобальные константы обычно объявляются в ВСЕ_ЗАГЛАВНЫЕ_БУКВЫ с подчеркиванием между ними.

например:

```
PI = 3.14159
GOOGLE_URL = "https://www.google.com"
```
