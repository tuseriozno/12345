Python немного отличается от других языков программирования тем, что у него нет блочной области видимости.

Это означает, что переменные, созданные вложенными в другие блоки кода, например, в циклах for, условных операторах if, циклах while и т.д., не получают локальной области видимости. Им присваивается функциональная область видимости, если они находятся внутри функции или глобальная область видимости, если они не находятся внутри функции.

например,

```
# Доступно везде
my_global_var = 1

def my_function():
    # Доступно только внутри my_function()
    my_local_var = 2
    
for _ in range(10):
    # Доступно везде
    my_block_var = 3

```
