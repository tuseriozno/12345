Вы можете заставить код разрешить вам изменять что-то с помощью global, если вы используете ключевое слово global перед его использованием.

например. Это вызовет ошибку

```
a = 1
def my_function():
    a += 1
    print(a)
```

Но это сработает
```
a = 1
def my_function():
    global a
    a += 1
    print(a)
```
