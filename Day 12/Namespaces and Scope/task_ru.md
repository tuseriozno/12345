### Локальная область видимости
Переменные (или функции), объявленные внутри функций, имеют локальную область видимости (также называемую областью видимости функции). Они видны только другому коду в том же блоке кода.

например:
```
def my_function():
    my_local_var = 2
    
# Это вызовет NameError
print(my_local_var) 
```

### Глобальная область видимости
Переменные или функции, объявленные на верхнем уровне (без отступа) файла кода, имеют глобальную область видимости. Она доступна в любом месте файла кода.

например:

```
my_global_var = 3

def my_function():
    # Это работает без проблем
    print(my_global_var)
```
